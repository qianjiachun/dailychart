Dailychart.js

Star

Tiny SVG charting library to display daily graph of a stock market security.

Examples

Default settings:

Dailychart.create('#chart-default');

Line width:

Dailychart.create('#chart-line', {
  lineWidth: 2
});

Line color:

Dailychart.create('#chart-color', {
  lineWidth: 2,
  colorPositive: '#3d3d3d',
  colorNegative: '#3d3d3d'
});

Transparent close line:

Dailychart.create('#chart-fill', {
  fillPositive: '#d6efda',
  fillNegative: '#fbdddd',
  closeColor: ''
});

Fill color:

Dailychart.create('#chart-fill', {
  fillPositive: '#d6efda',
  fillNegative: '#fbdddd',
  closeColor: ''
});

Intraday chart:

Dailychart.create('#chart-intraday', {
  fillPositive: '#d6efda',
  fillNegative: '#fbdddd'
});
